# \# Smart Restaurant Management System

# 

# \## Project Overview

# 

# The Smart Restaurant Management System is a cloud-native web application designed to modernize restaurant operations. The platform allows customers to browse menus, reserve tables, place food orders, and submit feedback, while administrators manage all restaurant activities through a secure admin dashboard.

# 

# \---

# 

# \## Features

# 

# \### 1. User Authentication

# \- Customer registration and login

# \- JWT-based authentication and authorization

# \- Role-based access control (Customer/Admin)

# \- Protected routes

# \- Secure password management

# \- User data stored in MongoDB

# 

# \### 2. Menu Management

# \- Responsive user interface built with React

# \- Browse available menu items

# \- Search and filter functionality

# \- Admin CRUD operations for menu management

# \- MongoDB-powered menu storage

# 

# \### 3. Order Management

# \- Place food orders online

# \- View order history

# \- Track order status

# \- Admin order status management:

# &#x20; - Preparing

# &#x20; - Ready

# &#x20; - Delivered

# \- State management using Redux

# 

# \### 4. Table Reservation

# \- Reserve tables by selecting date and time

# \- View reservation details

# \- Manage reservations

# \- Client-side and server-side validation

# \- MongoDB reservation storage

# 

# \### 5. Customer Feedback

# \- Submit ratings and reviews

# \- Display customer feedback

# \- Feedback management by administrators

# \- REST API-based CRUD operations

# 

# \### 6. Admin Dashboard

# \- User management

# \- Menu management

# \- Order management

# \- Reservation management

# \- Feedback management

# \- Secure admin-only access using JWT middleware
